Variants

Produce

Fields

name: String

An item’s name.

description: String

An item’s description.

Produces one item with a name and description and replies with its ID.

Transfers created NFT for an item to a producer.

Requirements

  • msg::source() must be a producer in a supply chain.

On success, returns SupplyChainEvent::Produced.

PutUpForSaleByProducer

Fields

item_id: ItemId

An item’s ID.

price: u128

An item’s price.

Puts an item up for a sale to a distributor for a given price on behalf of a producer.

Transfers item’s NFT to a supply chain.

Requirements

  • msg::source() must be a producer in a supply chain and a producer of this item.
  • Item’s ItemState must be Produced.

On success, returns SupplyChainEvent::ForSaleByProducer.

PurchaseByDistributor

Fields

item_id: ItemId

An item’s ID.

delivery_time: u64

A time in milliseconds for which a producer must deliver an item. A countdown starts after SupplyChainAction::ShipByProducer is executed.

Purchases an item from a producer on behalf of a distributor.

Transfers tokens for purchasing an item to a supply chain until an item is received (by SupplyChainAction::ReceiveByDistributor).

Note that an item’s producer must approve or not this purchase by SupplyChainAction::ApproveByProducer.

Requirements

  • msg::source() must be a distributor in a supply chain.
  • Item’s ItemState must be ForSaleByProducer.

On success, returns SupplyChainEvent::PurchasedByDistributor.

ApproveByProducer

Fields

item_id: ItemId

An item’s ID.

approve: bool

Yes (true) or no (false).

Approves or not a purchase from a distributor on behalf of a producer.

If a purchase is approved, then item’s ItemState changes to ApprovedByProducer and an item can be shipped.

If a purchase is not approved, then tokens for a purchase are refunded to an item’s producer.

Requirements

  • msg::source() must be a producer in a supply chain and a producer of this item.
  • Item’s ItemState must be PurchasedByDistributor.

On success, returns SupplyChainEvent::ApprovedByProducer.

ReceiveByDistributor(ItemId)

Tuple Fields

Receives a shipped item from a producer on behalf of a distributor.

Depending on a counted delivery time, transfers tokens for purchasing an item from a supply chain to a producer or as a penalty for being late refunds some or all of them to a distributor.

Transfers item’s NFT to a distributor.

Requirements

  • Item’s ItemState must be ShippedByProducer.

On success, returns SupplyChainEvent::ReceivedByDistributor.
